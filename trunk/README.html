README For Project Eyes-Free

1 Introduction

This directory contains source code for various eyes-free applications for the Android platform. It will also contain supporting code and libraries such as TTS.

2 Build Instructions

Each of these eyes-free applications can be built with a simple command-line of the form: 

ant -Dsdk-folder=<android-sdk-root>

Thus, to build the Talking Dialer, 

 cd talkingdialer
 ant-Dsdk-folder=<android-sdk-root>:
