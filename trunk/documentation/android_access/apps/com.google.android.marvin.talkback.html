Talkback

Description

TalkBack is a screen reader that provides spoken feedback when using native Android applications.

Usage:

1. Go to Settings -> Accessibility

2. Check the "Accessibility" and "TalkBack" checkboxes

About TalkBack:

-Does not run if accessibility is disabled

-Is part of the Android system and cannot be uninstalled

Notes

  • If you are on a Samsung device and are having trouble installing the update to Talkback, please do the firmware update for your device.
